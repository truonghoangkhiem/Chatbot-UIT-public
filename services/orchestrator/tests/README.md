# Tests Directory

This directory contains all test files for the orchestrator service.

## Files:

- `test_all_agents.py` - Comprehensive test suite for all agents
- `demo_orchestrator.py` - Demo script for orchestrator functionality

## Running Tests:

```bash
# Run all agent tests
python tests/test_all_agents.py

# Run orchestrator demo
python tests/demo_orchestrator.py
```
